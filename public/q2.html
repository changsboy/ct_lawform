문제 #2

숫자 N을 입력받아서넓이와 높이가N인 정사각형 형태의 그림을 그리면 됩니다. 문자'X'를이용하여 정사각형의 틀과 틀 안에 X자 그림을 그리고, 그 외 공간은 문자 'O'로 채우면 됩니다.

소스코드


let result = '';
for (let i = 0; i < N; i++) {
    let line = '';
    for (let j = 0; j < N; j++) {
        if (i === 0 || i === N - 1 || j === 0 || j === N - 1) {
            line += 'X';
        } else {
            if (i === j) {
                line += 'X';
            } else if ( i + j === N -1) {
                line += 'X'
            } else {
                line += 'O';
            }
        }
    }
    result += line + '\n';
}
